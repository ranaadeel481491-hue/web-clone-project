# 🏆 SportX | Gear Up

SportX ek modern sports e-commerce landing page hai jahan users premium sports accessories explore kar sakte hain. Yeh project basic frontend technologies (HTML, CSS, JS) use karke banaya gaya hai.

---

## 🚀 Features

* 🧭 Responsive Navigation Bar
* 🎯 Hero Section with Call-to-Action (Shop Now)
* 🛍️ Product Showcase (Grid Layout)
* 🧾 Clean Footer Section
* ⚡ Smooth Scroll Interaction (JavaScript)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
